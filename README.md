# 🌿 LeafGuard AI

AI Powered Plant Leaf Disease Detection System using Deep Learning, Computer Vision, Flask, and Streamlit.

---

## 📌 Overview

LeafGuard AI is an intelligent plant disease detection system that identifies crop leaf diseases using Deep Learning and Computer Vision.

The project supports:
- 🌱 Plant Leaf Disease Detection
- 🧠 AI-based Prediction System
- 📊 Confidence Analysis
- 🩺 Symptoms & Causes Identification
- 💊 Treatment Recommendations
- 🌿 Prevention Strategies
- 📥 Downloadable Disease Reports
- ✅ Invalid Image Detection (Non-leaf image validation)

This system helps farmers, researchers, and agriculture students detect diseases early and improve crop productivity.

---

# 🚀 Features

## ✅ AI Disease Prediction
Detects diseases from uploaded crop leaf images using Deep Learning.

## ✅ Leaf Validation System
Checks whether the uploaded image is actually a plant leaf before prediction.

## ✅ Detailed Disease Information
Provides:
- Symptoms
- Causes
- Prevention Methods
- Treatment Suggestions
- Severity Level
- Professional Farming Tips

## ✅ Professional UI
Modern responsive interface built using:
- Streamlit
- Flask
- Custom CSS

## ✅ Downloadable Reports
Generate and download disease analysis reports instantly.
